# [BitDiff](https://bitdiff.net)
A coin arbitrage lookup site by [Switch](https://twitter.com/SwitchLove) and [Mouse](https://twitter.com/MouseDoesCrypto).

### Setup
```npm i
npm run build
npm start

# development mode (watches for source changes)
npm run watch```

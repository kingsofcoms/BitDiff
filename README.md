# [BitDiff](https://bitdiff.net)
A coin arbitrage lookup site by [Switch](https://twitter.com/SwitchLove) and [Mouse](https://twitter.com/0x0298).

### Setup
```
npm i
npm run build
npm start

# development mode (watches for source changes)
npm run watch
```

### Roadmap
